Learning Goals

The general learning goals of this course are to:
  • Learn how to design basic machine learning algorithms.
  • Identify and understand specific, widely-used machine learning algorithms.
  • Learn how to apply machine learning algorithms to read data and evaluate the algorithms' performance.
More concretely, by the end of the semester, the students will be expected to have mastered fundamental concepts in machine learning including:
  • Derive analytical solutions for mathematical fundamentals of ML (partial derivatives, matrix multiplication, etc.)
  • Identify the relationship between regularization and training size on training and test accuracy
  • Derive and implement learning algorithms
  • Implement a backpropagation algorithm by hand
  • Understand and articulate bias-variance tradeoff
In regards to application of machine learning to a data science projects, students will leave the course able to:
  • Identify important machine learning problems
  • Verbally explain problems
  • Produce technically-proficient writing
  • Implement machine learning tools on real data and evaluate performance

Grading

  • Homeworks (4) — 40%
  • "Celebrations of learning" (2) — 20%
  • Project — 30%
  • Class engagement — 10%


The project grade (30%) is further broken down to:
  • Idea proposal — 2.5%
  • Paper selection — 2.5%
  • Literature review — 5%
  • Project plan — 5%
  • Final presentation (poster session) — 5%
  • Final report — 10%

Class engagement grades are a composite of non-graded assignments including but not limited to: (a) in-class worksheets, (b) posting on Piazza, (c) writing up scribe notes.

Homework deadlines are strict. Homework not turned in ontime will NOT be accepted.

There will be two ``celebrations of learning." These will be in-class, closed-book exams.
